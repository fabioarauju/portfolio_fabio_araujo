SafeRoute

HTML CSS

SafeRoute é um protótipo feito para o Global Solutions que deve ter continuidade ao longo do ano. É uma aplicação web criada para ajudar comunidades afetadas por enchentes, oferecendo mapas interativos das áreas de risco, rotas seguras de evacuação, localização de abrigos próximos e alertas em tempo real. O sistema é acessível, responsivo e inclui recursos de inclusão, como tradução para Libras. Nosso foco é salvar vidas, facilitando decisões rápidas e seguras em situações de emergência.
